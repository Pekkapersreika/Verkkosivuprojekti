  2. What's DOCTYPE declaration meant for?
    • Doctype stands for Document Type Declaration. It informs the web browser about the type and version of HTML used in building the web document. This helps the browser to handle and load it properly.

  4. How is it possible to define the text visible on the top of a browsers window and/or on a tab?
    How about adding an icon next to the title? Give examples.
    • Text is added to the top of a browser and/or on a tab by adding your text inside <title> element within the <head>
    • An icon can be added next to the text by adding a <link>, which has either an absolute or a relative URL in it, into the <head> element

  5. How do block-level and inline level elements differ from each other? Give examples.
    • The <div> element is a block element, and will always start on a new line and take up the full width available (stretches out to the left and right as far as it can).
    • The <span> element is an inline element, and will not start on a new line and only takes up as much width as necessary.

  1. There are three different ways of adding CSS styles on an (X)HTML page. How? What are the benefits of each way?
    • Inline
    • Inline styling affects only the element you apply it to
    • Not much benefits
    • Internal
    • Internal styling is embedded inside the <head> element and only apply to the page you are working on
    • I'd say the benefit is that you dont need a separate CSS file
    • External
    • External styles are used for the whole, multiple-page website and there is a separate CSS file
    • In my opinion the biggest benefits are: there is no long string of styling at the start of each page and that it applies the styling to all of the pages
